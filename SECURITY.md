# Security Policy

We take the security of **EdgeKits Core** seriously. If you believe you have found a security vulnerability in this project, please report it to us as described below.

## Supported Versions

Since this is a starter kit/boilerplate, we generally support only the latest version.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you have discovered a security vulnerability, please email us at **hello@edgekits.dev**.

Please include as much information as possible to help us reproduce the issue:

- The version of the starter kit used.
- Steps to reproduce the vulnerability.
- Any relevant code snippets or configuration.

We will review your report and respond within 48 hours to acknowledge receipt. We will prioritize the fix and release a patch or update instructions as soon as possible.

## Disclosure Policy

We ask that you give us a reasonable amount of time to fix the issue before making it public. In return, we will openly acknowledge your contribution and credit you (if you wish) in our release notes once the vulnerability is fixed.

Thank you for helping keep EdgeKits Core safe!
